# Financial-Loan-Analysis-Dashboard

Project Overview

The Financial Loan Analysis Dashboard is an interactive Power BI solution designed to analyze loan performance, customer demographics, and risk factors. The dashboard provides key insights into loan applications, funded amounts, customer income, loan grades, repayment behavior, and charged-off loans to support data-driven lending decisions.

The project is divided into three main analytical sections:

Overview Dashboard – High-level KPIs and loan portfolio performance.
Customer Insights Dashboard – Customer demographics, income distribution, employment analysis, and geographic trends.
Risk Analysis Dashboard – Loan default patterns, risk segmentation, charged-off loan analysis, and high-risk state identification.

Objectives
Monitor overall loan portfolio performance.
Analyze borrower demographics and financial profiles.
Identify high-risk customers and loan categories.
Track loan repayment and charged-off trends.
Support decision-making for risk management and lending strategies.
